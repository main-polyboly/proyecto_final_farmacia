# proyecto_final_farmacia

Proyecto final ISID223. 
Consultoría de digitalización para una cadena de farmacias de barrio (4 sucursales, 3 zonas), integrando ventas, inventario y finanzas en un solo flujo automatizado.
La farmacia operaba con procesos manuales, silos de información y sin trazabilidad entre ventas, inventario y finanzas. La propuesta digitaliza todo el ciclo: una venta actualiza automáticamente toda la cadena de valor, sin intervención manual.
